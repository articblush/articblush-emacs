<h2 align="center"> Articblush Emacs </h2> 

## Installation

run this command

```
wget https://raw.githubusercontent.com/articblush/emacs/main/articblush-theme.el -O ~/.emacs.d/articblush-theme.el
```
then run ```M-x customize-themes``` and select articblush.
